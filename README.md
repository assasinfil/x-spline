# x-spline

## Зависимости
- [openmp](https://www.openmp.org/)
- [matplotlib](https://pypi.org/project/matplotlib/)
## Компиляция
### Release
```bash
cmake -DCMAKE_BUILD_TYPE=Release .
make
```
### Debug
```bash
cmake -DCMAKE_BUILD_TYPE=Debug .
make
```
## Использование
### Запуск
```bash
./x-spline
```
### Визуализация
```python
python visualiser.py
```
